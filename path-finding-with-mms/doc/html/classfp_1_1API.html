Detailed Description

API class for interaction with mms simulator.

API class interacts with the mms simulator through std::cout and std::cin to send and receive command/information.
Member functions of this class are as follows:
mazeWidth(): mazeHeight(): wallFront(): wallRight(): wallLeft(): moveForward(): turnRight(): turnLeft(): setWall(): clearWall(): setColor(): clearColor(): clearAllColor(): setText(): clearText(): clearAllText(): wasReset(): ackReset():
